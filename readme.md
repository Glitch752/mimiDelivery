# Mimi Delivery
![Game Logo](https://raw.githubusercontent.com/Glitch752/mimiDelivery/refs/heads/main/art/MimiDeliveryLogo.png) \
You are Mimi - an axolotl that is hired by Mimo to be a delivery driver on the island of Singapurr. Your goal is to deliver Blahajs from IKEE and Durians to the customers! 

Made for [Overglade](https://overglade.hackclub.com/), irl game jam organized by Hack Club!
